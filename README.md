# 🚀 Churn Analysis with Automation

This project was developed to analyze customer churn behavior and demonstrate how Python can be used to quickly generate insights and automate reporting.

---

## 📊 Context

During the week, I faced a scenario where I needed to deliver insights in a very short time.  
Although tools like Power BI are great for dashboards, I explored how Python can also be used for fast and flexible data analysis and automation.

---

## 🎯 Objective

- Analyze customer churn data  
- Identify key patterns and risk factors  
- Generate visual insights  
- Automate report delivery via email  

---

## 🧰 Technologies Used

- Python  
- Pandas  
- Matplotlib  
- SMTP (Email Automation)  

---

## 🔍 Analysis Performed

- Churn rate by age  
- Churn rate by country  
- Customer behavior analysis  

---

## 📈 Key Insights

- Customers over 50 years old show higher churn rates  
- Germany has the highest churn rate among countries  
- Inactive customers are more likely to churn
